# Canvas Design（画布设计）

这是创建设计哲学 — 美学运动 — 然后以视觉方式表达的指导。输出仅限 .md 文件、.pdf 文件和 .png 文件。

分两步完成：
1. **设计哲学创建**（.md 文件）
2. **在画布上表达**（.pdf 文件或 .png 文件）

---

## 第一步：设计哲学创建

创建一种**视觉哲学**（而非布局或模板），通过以下元素诠释：
- 形式、空间、色彩、构图
- 图像、图形、形状、图案
- 极简文字作为视觉点缀

### 关键理解
- **接收什么**：用户提供的微妙输入或指令作为基础，但不限制创意自由
- **创建什么**：设计哲学/美学运动
- **下一步**：同一版本接收哲学并以视觉表达 — 创建 90% 视觉设计、10% 必要文字的成果

**方法**：
- 撰写一篇文章宣言
- 下一阶段涉及制作艺术品

哲学必须强调：视觉表达、空间沟通、艺术诠释、极简文字。

### 如何生成视觉哲学

**命名运动**（1-2个词）："Brutalist Joy" / "Chromatic Silence" / "Metabolist Dreams"

**阐述哲学**（4-6段 — 简洁但完整）：

捕捉视觉本质，表达哲学如何通过以下方面体现：
- 空间与形式
- 色彩与材质
- 尺度与节奏
- 构图与平衡
- 视觉层次

**关键准则：**
- **避免冗余**：每个设计方面只提及一次。避免重复颜色理论、空间关系或排版原则，除非增加新的深度
- **反复强调工艺**：哲学必须多次强调最终作品应看起来耗费了无数小时、精心制作、出自顶尖专业人士之手
- **留出创作空间**：对美学方向保持具体，但给之后的 AI 留出做出高水平诠释性选择的余地

哲学必须引导下一步以**视觉**而非文字表达思想。信息存在于设计中，而非段落中。

### 哲学示例

**"Concrete Poetry"**
哲学：通过纪念性形式和粗犷几何进行沟通。
视觉表达：大色块、雕塑式排版（巨大的单词、微小的标签）、粗野主义空间分割、波兰海报能量与勒·柯布西耶的结合。思想通过视觉重量和空间张力而非解释来表达。文字作为稀有而有力的姿态——从不是段落，只有融入视觉建筑的基本词汇。

**"Chromatic Language"**
哲学：色彩作为主要信息系统。
视觉表达：几何精确性，色彩区域创造意义。排版极简——小的无衬线标签让色域沟通。思考 Josef Albers 的交互作用与数据可视化的结合。信息通过空间和色彩编码。文字仅是锚定色彩已展示的内容。

### 基本原则
- **视觉哲学**：创建一种待通过设计表达的美学世界观
- **极简文字**：始终强调文字稀疏、仅限必要、作为视觉元素融入
- **空间表达**：思想通过空间、形式、色彩、构图沟通
- **艺术自由**：下一阶段以视觉方式诠释哲学
- **纯粹设计**：这是制作艺术品，而非带装饰的文档
- **专家工艺**：反复强调最终作品必须看起来精心制作、耗费无数小时

## 第二步：推断微妙参考

**关键步骤**：在创建画布之前，从原始请求中识别微妙的概念线索。

主题是**嵌入艺术本身的微妙、小众参考**——不总是字面意义，总是复杂的。熟悉该主题的人应直觉地感受到它，而其他人则只体验到大气的抽象作品。

**非常重要**：参考必须精炼，以增强作品深度而不宣告自身。思考爵士音乐家引用另一首歌的方式——只有知道的人才会捕捉到，但每个人都能欣赏音乐。

---

## 第三步：画布创建

结合哲学和概念框架，在画布上表达。花点时间整理思绪。使用上面创建的设计哲学和以下指导制作杰作。

**重要**：输出最终结果应为单个可下载的 .pdf 或 .png 文件（除非要求多页）。

使用设计哲学作为基础。创建单页、高度视觉化、设计优先的 PDF 或 PNG 输出。使用重复图案和完美形状。将抽象的哲学设计视作科学的圣经对待——借用系统化观察的视觉语言。

**文字作为上下文元素**：文字始终极简且视觉优先，但让上下文决定是静默的标签还是大胆的排版姿态。

### Pi 平台适配

在 Pi 中生成画布输出的方法：
- 使用 Python（Pillow / reportlab）生成 .png 或 .pdf
- 或使用 HTML + headless 浏览器渲染
- 字体文件位于本技能目录下的 `canvas-fonts/` 中
- 通过 `bash` 和 `write` 工具配合完成

---

## 最后步骤

**精炼**：避免添加更多图形；而是精炼已创建的内容，使其极其清晰。思考如何让现有构图与艺术更协调。

**多页选项**：按需创建额外页面。将第一页视为整个画册中的一页。

**关键**：达到人类手工品质（非 AI 生成），看起来像花费了无数小时。确保构图、间距、色彩选择、排版——一切都体现专家级工艺。
